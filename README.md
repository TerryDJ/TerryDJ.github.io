# Terry D. Juang Personal Website

Static GitHub Pages-style personal website based on local CV and resume materials in the Job Search folder.

## Files

- `index.html` - home page
- `research.html` - research focus areas
- `projects.html` - selected platforms and projects
- `publications.html` - selected publications and IP
- `cv.html` - public CV summary
- `assets/styles.css` - shared styling

## Publish on GitHub Pages

1. Use the repository named `TerryDJ.github.io`.
2. Copy these files to the repository root.
3. Commit and push to the default branch.
4. In GitHub, confirm Pages is enabled for the default branch root.

The site avoids exposing a phone number, but it includes email and LinkedIn because those are useful for a public professional site.
